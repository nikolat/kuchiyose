## Supported Event Kinds

- 0: User Metadata
- 3: Follows
- 5: Event Deletion Request
- 7: Reaction
- 17: Reaction to a website
- 1111: Comment
- 10000: Mute list
- 10002: Relay List Metadata
- 10006: Blocked relays list
- 10030: User emoji list
- 22242: Client Authentication
- 30030: Emoji sets
- 39701: Web bookmarks
